# Savirix Documentation – Apex27 Module Captures

This directory collates Scribe guides and notes for each Apex27 CRM module. Use the folder structure to jump into module-specific documentation and update the Feature Map as new guides are recorded.

## Structure
- **Dashboard & Navigation:** High-level navigation, dashboard widgets, and global search patterns.
- **Contacts Module:** Completed captures for list, creation, and detail views with data field notes.
- **Properties Module:** Completed captures covering marketing tabs, portal publishing, and key fields.
- **Tenancies Module:** Pending captures with required data points and workflow notes.
- **Payments Module:** Pending captures for rent processing and statement exports.
- **Calendar & Tasks:** Pending captures for event scheduling and task management.
- **Reports:** Pending captures for analytics and exports.
- **Settings & Workflows:** Pending captures for permissions, automations, and templates.
- **Marketing & Portals:** Pending captures for portal feeds and campaign tools.
- **Feature Map:** Summary status table for all modules.

## How to Update
1. Record the relevant workflow in Scribe using the format `Apex27 - [Module] - [Action]`.
2. Add the Scribe share link, capture date, and notes to the appropriate module README.
3. Update the Feature Map with the latest status emoji and notes.
4. Tag the dev team when modules reach ✅ Complete so implementation can proceed.
