## Data Field Notes
- **Contact Types:** Landlord, Tenant, Applicant, Contractor. Used for filtering and workflow triggers.
- **Primary Contact Details:** Name, email, phone numbers are mandatory for workflow automation.
- **Address Handling:** Utilises postcode lookup; manual override is permitted.
- **Compliance Fields:** ID verification date, compliance documents with expiry reminders.
