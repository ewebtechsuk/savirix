# Savirix Glossary

| Term | Definition |
| --- | --- |
| **Agent Company** | The estate agency or property management firm that purchases Savirix and operates its own workspace. May also be referred to as the **agency**. |
| **Agent User** | A member of an agent company. They log in to manage properties, contacts, and workflows within the agency workspace. |
| **Platform Admin** | Savirix platform staff responsible for provisioning agent companies, supporting onboarding, and maintaining system-wide configuration. |
| **Property Owner** | A landlord or vendor that owns or lets a property managed through an agent company. |
| **Property Renter** | A tenant or applicant renting or applying to rent a property managed by an agent company. |
| **Renter Portal** | Self-service tools used by property renters to track tenancies, requests, and statements. |
