## Data Field Notes
- **Rent & Pricing:** Supports weekly/monthly toggles with automatic annualised calculation.
- **Address & Geo:** Uses postcode lookup plus manual map pin adjustment.
- **Energy Performance:** EPC rating, certificate expiry, and document attachment.
- **Features:** Tag-based list used for website search filters; maintain consistent casing.
- **Portal Publishing:** Each portal has independent activation toggles and status indicators.
